## 导航结构

**主导航:** 基于角色的顶级导航，区分管理员、业务用户和最终用户视图

**次级导航:** 侧边栏导航，按功能模块组织，支持折叠和展开

**面包屑策略:** 层级面包屑显示当前位置，支持快速返回上级页面
